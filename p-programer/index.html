皮之三重境界

p-program Mar 24, 2020

古人曾经没有这么云过：

人之初，性本善，性相近，皮相远。

也就是说，人的皮性各有不同。有的不皮，有的很皮，更有甚者，皮成了仙。

古今之成大事业、大皮学家(p-programer)，罔不经过三种之境界：

昨夜西风凋碧树。独上高楼，皮尽天涯路

皮于表象，俗称皮皮虾。 皮皮虾经常出没于各大社群，常用句式有

  1. 21 天学会 C++
  2. PHP 是最好的语言
  3. C 语言从入门到放弃
  4. Java 是门啰嗦的语言
  5. 老子就是死也不优化JVM
  6. 机器学习真的不是统计学
  7. go 是云原生开发第一语言
  8. 没过几年就要跟微软学新语言
  9. 学 Python 要先买一把游标卡尺
  10. JavaScript 就是一把没有刀柄的剑
  11. 请不要把区块链当成收割韭菜的有效工具
  12. swift 是那个跟男朋友分手后就写歌的女歌手吗
  13. object-c 是被 Taylor Alison Swift 抛弃的备胎吗

衣带渐宽终不悔，为伊皮得人憔悴

经过第一重境界的轮回以后，皮皮虾顿悟了编程的奥义在于认清了口袋里的银子之后，依然要靠他们恰饭。

上善若皮，皮善覆万物而不争，处众人之所恶（wù），故几于道。

所以他化用了各家武功，形成了自己的独特武林绝学：医院挂号。 终不得治，谥号皮成仙。

众里寻他千百度，蓦然回首，那人皮在灯火阑珊处

皮成神，简称皮神，皮卡丘。

皮神是不属于人类的存在。它不呆在精灵球里，所以是不被束缚的存在。无法被观测，更无法被捕捉。其存在是一种传说。

皮即可爱，皮即正义，皮即世界。
